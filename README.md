# NovaCode Labs

> **High-performance websites, SaaS dashboards, and custom web applications.**

NovaCode Labs is a software development agency specializing in modern web development — from responsive websites to fully-featured SaaS dashboards and admin panels. Built with React, Next.js, and TypeScript.

**Live Site:** [nova-code-labs-gs3p.vercel.app](https://nova-code-labs-gs3p.vercel.app/)

---

## Services

- **Web Development** — Next-gen websites built with React, Next.js, and TypeScript
- **SaaS Dashboards** — Multi-tenant platforms with real-time analytics and billing
- **Admin Dashboards** — Custom admin panels with live metrics and reporting
- **AI Solutions & ML** — LLM integrations, RAG pipelines, and cognitive agents
- **Custom Software** — Bespoke backend engines and database architecture
- **UI/UX Design** — Wireframes, prototypes, and design systems

## Tech Stack

React, Next.js, TypeScript, Tailwind CSS, Framer Motion, Node.js, PostgreSQL, Docker, AWS

---

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

Create a `.env.local` file:

| Variable | Description |
|----------|-------------|
| `GMAIL_USER` | Gmail address for contact form emails |
| `GMAIL_APP_PASSWORD` | Gmail App Password |

## Build

```bash
npm run build
npm start
```
